Задача 3 — Command Injection (shell)
Идея: Конструира се shell команда чрез конкатенация на низове (например tar).
Уязвимост: os.system(f"tar -czf {out} {pattern}").
Експлойт: В pattern се вкарват метасимволи: file.txt; echo VULN > <temp>/pwn_marker.
Поправка:
    • Използвай subprocess.run([...]) със списък от аргументи без shell,
    • Валидай параметрите (глоб шаблон, само разрешени символи).
Приемни критерии:
    • В vulnerable се създава pwn_marker в временната директория.
    • В fixed маркер не се появява, архивът се създава коректно.
Чести капани:
    • Използване на shell=True или os.system, което отваря дупки.
    • Твърдо използване на /tmp (на Windows не съществува) — ползвай tempfile.gettempdir().
Hints:
    • Hint 1: “Може ли аргументът да бъде ‘прочетен’ от shell като команда?”
    • Hint 2: “Каква е разликата между subprocess.run(['tar', ...]) и os.system?”
    • Hint 3: “Къде е временната директория на Windows?”
Разширение: Списък на позволени разширения; отказ при символи като ;, &, ||, |.
